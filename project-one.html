About this project

Scaryplaces.com is one of my personal projects that is currently being developed. I was inspired to start this project when I couldn't find any websites dedicated to finding scary places.

This project is coded completely from scratch. All design and development was done by me. My biggest challenge was the carousel since I was new to JS. I am currently developing the location finder component that will search databases to find and display scary places for the user.
