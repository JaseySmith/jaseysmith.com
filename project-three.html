About this project

This is my static portfolio project. I kept the design simple without excluding the important elements of a web developer portfolio website. The hexagons I thought added a unique feature to the simple design.

I would have kept the website on one page but I wanted the user to have a full case study of each project. I'm proud to say my contact form is built with php on the backend. As a front-end developer it was a struggle to finally get it working, but just another problem to solve.
